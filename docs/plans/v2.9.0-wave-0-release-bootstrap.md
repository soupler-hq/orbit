# v2.9.0 Wave 0 Release Bootstrap

> Milestone: v2.9.0 — Idea to Market
> Phase: Wave 0
> Planned: 2026-03-31
> Final status: Implemented

## Phase Goal

Establish the operational infrastructure Orbit needs before it can self-orchestrate its own v2.9.0 release. Wave 0 delivers four capabilities:

1. **Observability** — structured status blocks so operators can see what Orbit is doing (#94, #110)
2. **Context discipline** — mid-session reload, scope boundary docs, orbit:ask command (#95, #96, #109)
3. **Unified installer** — single `orbit setup` entry point for all coding agents (#99)
4. **Persistent shared state** — SQLite context.db replacing STATE.md full-file reads (#100, #101)

Wave 0 is the prerequisite for Wave 1 (new agents) and all subsequent self-orchestration waves. **Nothing in v2.9.0 Waves 1–4 begins until Wave 0 ships.**

## Scope

- Observability and next-step guidance
- Context discipline and mid-session recovery
- Unified setup/install flow
- Shared structured state through `context.db`

## Non-Goals

- Wave 1 new agents (#64-#67)
- v3.0.0 features
- unrelated product-layer enhancements
